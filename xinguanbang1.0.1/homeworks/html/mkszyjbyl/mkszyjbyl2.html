马克思主义基本原理-2
2.1
1.[单选题]物质的唯一特性是( ) 答案：客观实在性
2.[单选题]唯物主义最初的历史形态是( ) 答案：朴素唯物主义
2.2
1.[单选题]“观念的东西不外乎是移入人的头脑并在人脑中改造过的物质的东西而已”。这就是说( ) 答案：意识是客观存在的主观映象
2.[单选题]“意识的能动性”或“主观能动性”,就是指( ) 答案：意识对于人们改造物质世界的指导作用
2.3
1.[单选题]“修昔底德陷阱”是指一个新崛起的大国会挑战现存大国,而现存大国也必然会回应这种威胁,这样冲突变得不可避免,这强调的是矛盾的斗争性。下列选项中与“修昔底德陷阱”不一样的是( ) 答案：合作共赢
2.[单选题]两个国家之间的摩擦和冲突,在一定条件下有可能转变为合作,这体现的是( ) 答案：矛盾的同一性
2.4
1.[单选题]下列选项中,没有体现矛盾的特殊性的是( ) 答案：大会漫灌式的扶贫
2.[单选题]矛盾普遍性和特殊性的关系是( )的关系 答案：共性和个性
2.5
1.[单选题]质和事物具有( )的关系 答案：直接同一
2.[单选题]“滴水穿石”蕴含的哲理是( ) 答案：量变引起质变
2.6
1.[单选题]下列成语中包含辩证否定原理的有( ) 答案：古为今用,洋为今用
2.[单选题]作为辩证否定的"扬弃"是( ) 答案：既克服又保留
2.7
1.[单选题]哲学的基本问题是( ) 答案：物质和意识的关系问题
2.[单选题]意识的本质就在于它( ) 答案：是反映物质世界的人脑的机能
3.[单选题]下列观点中表述唯物辩证法总特征的是( ) 答案：联系和发展的观点
4.[单选题]广州人在饮食上喜欢追求新异,对菜肴的名称更是讲究.前几年桌桌流行"发财"(发菜),现在则是逢酒席必上"生财"(生菜).吃发菜以求"发财",吃生菜以求"生财".从哲学上看这种做法否定了( ) 答案：事物的联系是现实的、具体的
5.[单选题]下列选项中,体现适度原则的是( ) 答案：过犹不及,适可而止
6.[单选题]矛盾的斗争性是指( ) 答案：矛盾双方相互排斥
7.[单选题]有这样一道数学题:"90%×90%×90%×90%×90%=?"其答案是约59%、90分看似一个非常不错的成绩,然而,在一项环环相扣的连续不断的工作中,如果每个环节都打点折扣,最终得出的成绩就是不及格.这里蕴含的辩证法道理是( ) 答案：量变引起质变
8.[多选题]物质是( ) 答案：标志着客观实在的哲学范畴；不依赖人的感觉的客观实在；各种具体物质形态的共性
9.[多选题]在意识能动性问题上,下列哪些判断是正确的( ) 答案：意识能动性的发挥必须以尊重客观规律为基础；意识对客观世界的反作用是通过实践实现的
10.[多选题]"勿以恶小而为之,勿以善小而不为".这句话体现的哲学道理有( ) 答案：事物的变化是从量变开始的；量变积累到一定程度就会引起质变
